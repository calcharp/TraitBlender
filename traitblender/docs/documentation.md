# Documentation Overview

Welcome to the comprehensive TraitBlender documentation. This section provides detailed information about using TraitBlender for generating museum-style morphological images.

## Documentation Structure

The documentation is organized into three main areas to serve different user needs:

??? info "User Interface & Workflows"
    **For users who prefer graphical interfaces and guided workflows**
    
    - **GUI Overview**: Complete interface walkthrough
    - **Panel Reference**: Detailed explanation of all UI elements  
    - **Workflow Guide**: Step-by-step processes for common tasks
    - **Configuration**: Settings and customization options
    
    **Best for**: Beginners, occasional users, those who prefer visual workflows

??? abstract "User API & Scripting"
    **For users who want programmatic control and automation**
    
    - **Core Classes**: Fundamental TraitBlender objects and methods
    - **Scene Assets**: Camera, lighting, and environment controls
    - **Transforms**: Morphological transformations and positioning
    - **Datasets**: Batch processing and data management
    - **CLI Tools**: Command-line utilities for automation
    
    **Best for**: Researchers with coding experience, batch processing, automation

??? gear "Development & Advanced Features"
    **For developers and contributors to TraitBlender**
    
    - **Architecture Overview**: How TraitBlender is structured
    - **Adding Custom Transforms**: Extend functionality
    - **Plugin Development**: Create TraitBlender extensions
    - **Contributing Guide**: How to contribute to the project
    - **Advanced Configuration**: Performance tuning and customization
    
    **Best for**: Developers, power users, contributors
