# Morphospaces

TraitBlender currently includes two morphospaces:

- **Shell (Default)** – a 3D shell model based on Contreras-Figueroa & Aragón (2023)
- **Circle Grid** – a white cube with a 4×4 grid of black circles on top

Each morphospace has a **name** (e.g. "Shell (Default)", "Circle Grid") used in the GUI and in config files, and:

- **traits** – columns in the dataset (per-specimen values)
- **hyperparameters** – extra settings that affect how the model is generated (set in the Configuration panel)

Use the pages below to see the full list of parameters, their default values, and the ranges that make sense in practice:

- [Shell (Default)](shell.md)
- [Circle Grid](circle-grid.md)
